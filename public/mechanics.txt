Your level is the minimum value of A and B. For example, if B = 0 and A = 10, then you would be at level 0 because the minimum value of 0 and 10 is 0.
The + button corresponding to A adds 1 to A.
The - button corresponding to A subtracts 1 from A.
The + button corresponding to B adds 1 to B.
The - button corresponding to B subtracts 1 from B.
The + and - buttons now swap places when one of them is clicked.
The “Switch” button can change the values of A and B.
There is now a 30 second timer which will set the values of A and B to negative when it reaches zero.
